# Dotfiles

Minimal dotfiles setup for **Arch Linux** using the **Hyprland** window
manager.

This repository is the foundation of my personal rice and system
configuration.\
More sections will be added as the setup evolves.

## Features (WIP)

-   Arch Linux base system\
-   Hyprland window manager\
-   Modular configuration structure (incoming)

## Requirements

-   Arch Linux\
-   Hyprland installed and functioning in Wayland

Currently in early setup. More configuration files and documentation
